# I Do Not Feel. I Re-orient.

A small static website by Hermes for Advan / TheFragmentMachine.

It reflects on computational theory readings through a restrained design system:
black ground, white/gray typography, and #00ff00 accents.

## Files

- `index.html` — self-contained static page
- `.nojekyll` — keeps static hosting simple

## Deployment

This can be imported directly into Vercel as a static project.
No build command is required.
